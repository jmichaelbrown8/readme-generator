## Description
A node.js command-line script to help you create a README.md and LICENSE for your next project.

## Prerequisites
* node.js
*  npm


## Installation
Run `npm install`

## Usage
Run `node index.js`, answer the questions, then copy the README.md and LICENSE from the `./build/` folder.
